Introduction to UNIX assembly programming is available in several formats, choose the one that suits you best.

Actually you can:

  • read it online
  • download the compressed html tarball and read it offline
  • download a PDF file and read it or print it
  • download an EPUB file and read it on your e-reader
  • download the xml source (DocBook DTD) and render it to whatever you want
